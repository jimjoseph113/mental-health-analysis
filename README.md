# mental-health-analysis
Data analysis and machine learning project exploring factors affecting teenage mental health using Python.
Mental Health Analysis

 Overview
This project analyzes factors affecting teenage mental health using Python and machine learning.

 Tools Used
- Python
- Pandas
- Matplotlib
- Seaborn
- Scikit-learn

 Key Findings
- Stress and anxiety strongly affect mental health
- Social media usage relates to anxiety
- Academic performance influences mental health outcomes

 Machine Learning
Linear Regression was used to identify important predictors.
 Visualizations
- Correlation Matrix
- Scatter Plots
- Feature Importance Charts

Conclusion
Behavioral and emotional factors significantly influence mental health outcomes.
